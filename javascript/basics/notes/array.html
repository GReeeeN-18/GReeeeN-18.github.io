    # JavaScript Array slice() 方法

    # 定义

    slice() 方法以新的数组对象,返回数组中被选中的元素

    # 语法

    array.slice(start, end)
    1

    # 参数值

    • start 可选，如果省略，则类似于"0"。
    • end 可选，如果省略，将选择开始位置到数组末尾的元素。使用负数从数组末尾进行选择。

    # 解释说明

    • slice()返回新的数组对象，返回被选中的元素（不改变原数组）
    • slice() 选择 start 参数开始的元素，并在给定的 end 参数处结束，但并不包括结束位

    # 代码演示

    let a = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
let b = a.slice(1, 3);
let c = a.slice(1);
let d = a.slice(1,-1);
let e = a.slice(-3,-1);
console.log(a); //Array(5) [ "Banana", "Orange", "Lemon", "Apple", "Mango" ]
console.log(b); //Array [ "Orange", "Lemon" ]
console.log(c); //Array(4) [ "Orange", "Lemon", "Apple", "Mango" ]
console.log(d); //Array(3) [ "Orange", "Lemon", "Apple" ]
console.log(e); //Array [ "Lemon", "Apple" ]
